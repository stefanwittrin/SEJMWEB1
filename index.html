What is Claude Code?

Claude Code is an agentic coding tool built by Anthropic. Unlike a chat interface, it runs as a CLI inside your terminal and has direct access to your file system, your git history, and your shell — so it can actually do things, not just suggest them.

🗂️

Reads your codebase

Claude Code explores your project files, understands structure, and uses that context when writing or editing code.

✏️

Writes & edits code

It creates new files, edits existing ones, and refactors code — all without you leaving the terminal.

🔧

Runs commands

Runs tests, installs packages, starts servers, and executes git commands on your behalf.

How it works

  1. 1

    You describe what you want

    Type a plain-English task in the terminal — "add a login page", "fix the failing test", "refactor this module".

  2. 2

    Claude explores the context

    It reads relevant files, checks git history, and builds a full picture of your project before doing anything.

  3. 3

    It acts — and shows you what it did

    Claude makes the changes, runs commands, and displays a diff so you stay in full control of every edit.

Key features

Slash commands

Built-in shortcuts like /clear, /review, and /init for common workflows.

MCP servers

Connect external tools — databases, APIs, browsers — so Claude can interact with your whole stack.

CLAUDE.md

A project memory file. Write down conventions and Claude will follow them in every session.

Git-aware

Reads your git log and diffs to understand what changed and why — useful for reviews and debugging.

IDE integration

Available as a VS Code and JetBrains extension so you can run it without leaving your editor.

Permission model

You control what Claude is allowed to run. Sensitive commands require your explicit approval.

Get started

Claude Code requires an Anthropic account. Install it with one command:

npm install -g @anthropic-ai/claude-code

Then open a terminal in any project folder and run:

claude
